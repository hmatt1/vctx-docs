title: Compiler Internals description: Covers 'ast', 'regen', 'tokens', and other developer-focused commands.Developer ToolsThese commands are for debugging the vctx compiler or understanding how it parses code.vctx ast file.vctx: Print the Abstract Syntax Tree.vctx tokens file.vctx: Show the stream of tokens from the lexer.vctx symbols file.vctx: Dump the symbol table for a file.vctx regen: Regenerate the standalone Lark parser (required if modifying vctx.lark).Resolution DebuggingYou can inspect how the compiler resolves types and assignments:python vctx-cli.py resolve-type counter Blinky examples.blinky
python vctx-cli.py assignment --ident counter --scope Blinky --package examples.blinky
